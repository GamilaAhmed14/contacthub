# ContactHub

A responsive contact management web application built with HTML5, CSS3, JavaScript, and Bootstrap.

This project focuses on building a complete CRUD application while practicing JavaScript fundamentals such as DOM manipulation, objects, arrays, Local Storage, searching, validation, and dynamic UI updates.


## Live Demo
https://contacthub-iota.vercel.app/

## Features

* Create new contacts
* Update existing contacts
* Delete contacts
* Search contacts by name, phone number, or email
* Store contacts using Local Storage
* Data persistence after page refresh
* Mark contacts as Favorite
* Mark contacts as Emergency Contacts
* Favorite Quick Access section
* Emergency Quick Access section
* Contact statistics dashboard
* Responsive design
* Bootstrap components
* Modal-based contact form
* Real-time form validation
* Semantic HTML structure
* BEM naming convention

## JavaScript Highlights

The application stores all contacts in an array of JavaScript objects and synchronizes the data with Local Storage to preserve it between sessions.

The application includes functionality to:

* Add new contacts
* Edit existing contacts
* Delete contacts
* Display all contacts dynamically
* Search contacts by name, phone number, or email
* Save and retrieve contacts from Local Storage
* Count and display the total number of contacts
* Count favorite contacts
* Count emergency contacts
* Generate Favorite Quick Access cards automatically
* Generate Emergency Quick Access cards automatically
* Toggle between Create and Update modes
* Reset the form after submission
* Open and close the contact modal
* Validate name, phone number, and email using Regular Expressions (Regex)
* Update the user interface instantly after every CRUD operation without refreshing the page
